Features

  • Real-time Shop Tracking - View current shop prices and inventory
  • Price History Charts - Track price trends over time
  • Waystone Directory - Find waystones by location and owner (currently quite buggy)
  • Multi-dimensional Support - Track items across Overworld, Nether, and End
  • Search & Filter - Quickly find specific items or waystones

How to Use

For Shops:

  • Browse the Shops page to see all available items
  • Use the search bar to find specific items
  • Click "Show History" to view price trends
  • Sort by any column to organize the data

For Waystones:

  • Visit the Waytones page to see all waystone locations
  • Search by waystone name or owner
  • View coordinates and dimensions for easy navigation

Contribute

Help keep the data up to date by using the ASMP Shop Get mod!

Download it from Modrinth to automatically collect and submit shop data from the ASMP server.

Need help or want to report issues? Contact us on Discord!

Technical Details

This website is built using Cursor and Node.js. The data is stored in plain json files. There are API routes if you want to use the data for yourself (/asmp/api/shops or /waystones).
